# lazydmi

A simple CLI tool for reading SMBIOS/DMI tables using [lazybios](https://github.com/LazySeldi/lazybios).

## Installation

### From Source

```bash
git clone https://github.com/LazySeldi/lazydmi
cd lazydmi
make
sudo make install
```

### Arch Linux
```bash
yay -S lazydmi #or your preferred AUR package manager

```
